# 🚀 Multi-Tools Hub

Um dashboard interativo de ferramentas utilitárias desenvolvido com **React.js** e **Tailwind CSS**. O projeto foca em uma interface moderna utilizando a estética _Neomorphism_ e uma arquitetura escalável voltada para a experiência do usuário.

---

## 🛠️ Tecnologias e Conceitos

Este projeto foi construído para consolidar conhecimentos fundamentais de Engenharia de Software no ecossistema Frontend:

- **Core:** [React.js](https://reactjs.org/) (Hooks, Composition, State Management)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/) para design responsivo e efeitos de vidro.
- **Animações:** [Framer Motion](https://www.framer.com/motion/) (Efeitos de cascata, layout transitions e feedback tátil).
- **Roteamento:** [React Router Dom](https://reactrouter.com/) para navegação entre ferramentas.

---

## 📱 Ferramentas Disponíveis

O hub centraliza cinco aplicações principais, cada uma explorando um desafio técnico diferente:

- **📝 To-Do List:** Gestão de tarefas com persistência em `localStorage` e animações de remoção suave.
- **🔢 Click Counter:** Contador interativo com micro-interações e sistema de metas.
- **🎮 Jogo da Velha:** Lógica de jogo PvP (Player vs Player) com detecção de vitória e empate.
- **🧮 Calculadora:** Operações matemáticas básicas com interface otimizada e tratamento de erros.
- **📍 Buscador de CEP:** Consumo de API externa ([ViaCEP](https://viacep.com.br/)) com tratamento de estados assíncronos (loading/error).
